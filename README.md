# dashboard_demo

## 簡介
使用 Sing App 模板及圖表等套件，練習製作簡易 dashboard。

## 主要技術
* Vue
* Vuex
* bootstrap
* bootstrap-vue
* highcharts
* xterm

## 說明
### vue-styleguidist
開啟樣式導覽服務：

`yarn styleguide`

執行編譯靜態版本：

`yarn styleguide:build`

## 聲明
採用模板：[Sing App](http://https://github.com/flatlogic/sing-app-vue-dashboard "Sing App")
